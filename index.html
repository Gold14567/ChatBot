ChatBot

Commands

There are four permission levels (permlvl for short)

  • Permlvl 1 - A Normal Member
  • Permlvl 2 - A Moderator
  • Permlvl 3 - An Admin
  • Permlvl 4 - The Server Owner
  • Permlvl 5 - Me
You will need to put the bots prefix in front of all the commands. The default prefix is ; but your server admins can change this. <field> marks a required field and [field] marks an optional field

Main

Name Description Usage Permlvl Example
Help Shows you a list of commands help 1 ;help
Ping Gives you the bots ping in milliseconds ping 1 ;ping
Time Shows the bots time, useful for setting the msgTimer setting time 1 ;time

Fun

Name Description Usage Permlvl Example
8ball Gives you a random message from an 8ball 8ball 1 ;8ball
Random Generate a random number between 2 numbers random <num1> <num2> 1 ;random 5 10
Random Fact Gives you a random fact. randomfact 1 ;randomfact
